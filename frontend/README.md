# Online Teaching School Management System

## 🚀 Features

- **Role-based Dashboards** (Admin, Teacher, Student)
- **Live Classes with Custom WebRTC**
- **Homework System with Image/Audio Upload**
- **Auto Cleanup Rule (6 interactions → keep latest 2)**
- **Leave Request & Auto Replacement System**
- **Real-time Notifications**
- **Analytics Dashboard**
- **Report Export (PDF, Excel, CSV)**
- **Payment-based Video Access**

## 🛠️ Tech Stack

- **Frontend**: React + Vite + TailwindCSS + DaisyUI
- **Backend**: Node.js + Express + Socket.IO
- **Database**: MySQL
- **Real-time**: WebSockets
- **Video**: WebRTC (Custom implementation)
- **File Storage**: Cloudinary
